Portfolio project

Doel van het project

Een portfoliopagina vormgeven en bouwen die laat zien wat ik tot nu toe geleerd heb. Ik probeer de toegankelijkheidseisen zoveel mogelijk mee te nemen in het proces.

Proces

In Figma heb ik een wireframe gemaakt voor kleine schermen (telefoon) en grote schermen (desktop/laptop). De foto's op de website komen uit mijn eigen fotobibliotheek. Ik heb deze allemaal zelf gemaakt in de afgelopen jaren. De website is gebouwd in Html en CSS met een beetje JavaScript voor de functionaliteit van het navigatiemenu.

Eindresultaat

Een portfolio website waar ik blij mee ben, die bij mij past, laat zien waar ik voor sta en ga en die laat zien wat ik tot nu toe geleerd heb. Uiteraard is dit een ongoing proces en is de website nooit helemaal af.
