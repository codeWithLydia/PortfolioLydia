Eindproject React

Doel van het project

Dit project is de laatste opdracht van m'n opleiding tot front-end developer. Het eindresultaat moest voldoen aan bepaalde eisen waarbij het design iets minder belangrijk was. Het ging voornamelijk om de code en de functionaliteit van de webapp.

Proces

Om te voldoen aan alle eisen en om niks te vergeten, heb ik als eerste een lijst gemaakt met alle onderdelen en eisen. Daarna heb ik een wireframe gemaakt in Figma. Zo had ik een leidraad voor het eindresultaat. De webapp heb ik gebouwd in VS code. Hiervoor heb ik gebruik gemaakt van ReactJS, want dit was 1 van de eisen.

Resultaat

Een webapp die aan alle eisen voldoet, die responsive is en die overzichtelijk is voor de gebruiker.
